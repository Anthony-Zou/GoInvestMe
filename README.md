## Getting Started
1. Clone the repository
2. Run `npm install`
3. Copy `.env.example` to `.env` and configure
4. Run `npm run compile` to build contracts
5. Run `npm run test` to verify everything works

For detailed setup instructions, see `docs/README.md`.
